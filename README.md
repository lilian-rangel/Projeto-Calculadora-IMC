<h2>Como utilizar a calculadora</h2>
<ul>
<li>Adicione sua altura, conforme exemplo.</li>
<li>Adicione seu peso, conforme exemplo.</li>
<li>Clique no botão <b><i>Calcular</i></b>.</li>
<ul><li>Caso tenha colocado as informações de forma incorreta, basta clicar no botão <b><i>Limpar</i></b>.</li></ul>
<li>Aparecerá o resultado do seu IMC e a sua classificação.</li>
<li>Caso queira calcular novamente, basta clicar no botão <b><i>Voltar</i></b>.</li>
</ul>
